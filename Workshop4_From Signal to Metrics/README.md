# Workshop 4 — From Signal to Metrics

![Framework Banner](../docs/wearables_Galileo.png)

## Different instruments. Same challenge.

From Galileo’s sparse observations of Jupiter to modern wearable sensing, the scientific challenge remains surprisingly similar:

> How do incomplete observations become meaningful claims?

A smartphone does not directly measure fatigue, stress, stability, or recovery.

It measures signals.

This workshop explores the transition from:

signal → metric → interpretation → claim

using movement data, Python notebooks, and exploratory analysis.

---

> A wearable does not only measure signals.  
> It makes claims about the body.
